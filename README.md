# UniStyle

A modern e-commerce web application for clothing and apparel, built with the MERN stack. Customers can browse collections, view product details, add items to cart, and place secure orders â€” while admins manage inventory, products, promotions, and users efficiently.

---

##  Features

###  Customer (Frontend)

- Browse product collections by category  
- View detailed product pages (images, sizes, reviews, stock info)  
- Add items to cart & update quantities  
- Secure checkout flow  
- Create an account / Login  
- View past orders  

###  Admin Dashboard

- Product management (Create / Read / Update / Delete)  
- Inventory & category control  
- Set discounts and promotions  
- User management  
- Audit logs & activity tracking  

---

##  Tech Stack

###  Frontend

- React + Vite  
- React Router  
- Context API (global state)  
- Custom CSS / Tailwind-like styling  
- Deployed via Render  

###  Backend

- Node.js / Express  
- MongoDB + Mongoose  
- JWT Authentication  
- Deployed via Render

##  Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/unistyle-comp231/unistyle.git
cd unistyle
```

### 2. Install frontend & backend dependencies

```bash
cd frontend
npm install

cd ../backend
npm install
```

### 3. Start development servers
**Frontend**
```bash
cd frontend
npm run dev
```

**Backend**
```bash
cd backend
npm start
```

## Deployment
- Frontend and backend deployed on Render
- MongoDB Atlas used as the production database
